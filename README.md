# My-Kindle-Notes

A beautiful, responsive website for viewing your Kindle highlights and notes.

## Features

- Clean, responsive design that works on desktop and mobile devices
- Card-based navigation for easy access to all your notes
- Search functionality to quickly find specific books
- Alphabetical filtering to browse by first letter
- Individual pages for each book's highlights with easy navigation back to home

## Setup Instructions

1. Unzip the contents of this file
2. Upload all files to your GitHub repository named "My-Kindle-Notes"
3. Enable GitHub Pages in your repository settings:
   - Go to Settings > Pages
   - Select "main" branch as the source
   - Click Save

Your website will be available at: https://[your-username].github.io/My-Kindle-Notes/

## Structure

- `index.html` - Homepage with navigation to all notes
- `css/` - Stylesheets for the website
- `js/` - JavaScript functionality
- `notes/` - Individual HTML pages for each book's highlights
- `assets/` - Additional resources

## Customization

You can customize the website by modifying the CSS files:
- `css/style.css` - Main styling
- `css/mobile.css` - Mobile-specific styling
- `css/responsive.css` - Responsive design elements
- `css/note-page.css` - Styling for individual note pages
